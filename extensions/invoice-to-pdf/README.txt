All steps are required.  As always, backup your DATABASE and OPENCART files!!!

1. Place the 'system' folder in OpenCart via FTP.  No files are overwritten.

2. Place the 'vqmod' folder in OpenCart via FTP.  No files are overwritten.

3. VERY IMPORTANT: You must do a chmod 777 on the upload\system\library\dompdf\lib\fonts folder!

4. Test the download of an order.
